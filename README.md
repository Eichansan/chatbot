# テスト

## 事前準備
以下のコマンドをインストールしてください。(必要なコマンドがあればリンクを追記してください)
- [Taskfile](https://taskfile.dev/installation/)
<!--- - [Taskfile](https://taskfile.dev/ja-JP/installation/) --->

## インストール

```bash
task install
```

## 起動

```bash
task up
```

## 実行

- app/wiki下にexample.txtを配置
- Chainlit（http://localhost:3000）で実行するだけ 

## 停止
```bash
task stop
```

## 削除
```bash
task down
```
